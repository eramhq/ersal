# پول

تجارت ایرانی ریال و تومان را به جای هم به کار می‌برد. ۱ تومان = ۱۰ ریال. API شرکت‌های حمل معمولاً ریال می‌خواهند، ولی فروشنده به تومان فکر می‌کند. خطای ۱۰ برابر در هر جهت فاجعه است.

ارسال این را با value object `Amount` حل می‌کند:

```php
use Eram\Ersal\Money\Amount;

$cost = Amount::fromToman(50_000);  // فروشنده به تومان فکر می‌کند
$cost->inRials();                    // ۵۰۰,۰۰۰ — API شرکت ریال دریافت می‌کند
$cost->inToman();                    // ۵۰,۰۰۰ — نمایش به تومان
```

`Amount` همه چیز را داخلی به ریال ذخیره می‌کند. هر provider می‌داند API خودش چه واحدی می‌خواهد و به طور خودکار تبدیل می‌کند.

## ساخت

```php
Amount::fromRials(500_000);  // ۵۰۰,۰۰۰ ریال = ۵۰,۰۰۰ تومان
Amount::fromToman(50_000);   // ۵۰,۰۰۰ تومان = ۵۰۰,۰۰۰ ریال
```

مقدار منفی `InvalidAmountException` پرتاب می‌کند.

## عملیات ریاضی (تغییرناپذیر)

```php
$a = Amount::fromToman(10_000);
$b = Amount::fromToman(3_000);

$c = $a->add($b);        // ۱۳,۰۰۰ تومان (Amount جدید)
$d = $a->subtract($b);   // ۷,۰۰۰ تومان (Amount جدید)
// $a تغییر نمی‌کند
```
